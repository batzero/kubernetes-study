#### 1. pod

- 한 개 이상의 컨테이너로 구성됨. 

- 가장 작은 deploy  단위 

  

##### pod 생성 

1. 아래와 같이 yaml 파일 작성 

<pod-test.yaml>

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: pod-test
spec:
  containers:
  - name: nginx
    image: nginx
  - name: centos
    image: centos
    command: ["/bin/sh", "-c", "while : ;do curl http://localhost:80/; sleep 3; done"]
```



2. 명령어 입력 

```
kubectl create -f pod-test.yaml 
```
